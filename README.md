# xbackup

Export/Backup all IGC files you have uploaded to DHV-XC

![Alt text](xcbackup.jpg?raw=true "Screenshot")

# usage
```
Usage:
  xcbackup [OPTIONS]

Application Options:
  -u, --user= DHV-XC User name
  -p, --pass= DHV-XC User Password
  -d, --dir=  Target directory

Help Options:
  -h, --help  Show this help message

```
